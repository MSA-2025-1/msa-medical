# 슬기로운 EMR

소규모 병원에서 사용하는 <b>통합 예약 / 진료 관리</b> 시스템

## 프로젝트 소개

- <b>예약부터 진료, 문서 발급, 결제</b>까지 One-Stop으로 가능한 소규모 병원용 통합 EMR 시스템 개발
- 의사 2명, 간호사 2명의 소규모 인원을 상정
- 의료진 간에 채팅도 가능

## 개발 목적

- JavaScript와 Vue.js를 사용한 SPA 클라이언트 개발
- Java와 Spring Boot를 사용한 REST API 방식의 서버 개발
- 웹소켓을 이용한 실시간 메세지 전달
